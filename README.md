# pets-up
基于flask，记录宠物的成长

# 项目框架
common -    constants   通用常量
            dao         数据库查询
            error       错误码
            initializations     初始化数据
            models      模型定义
            utils       通用工具

core - admin  后台管理
              api         接口
              settings    后台配置

# 项目启动
执行run-web.py
